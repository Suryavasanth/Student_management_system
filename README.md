# Student Management System

---

## Features

- User Signup & Login  
- JWT Authentication  
- Protected Routes  
- Add Student Details  
- View Student List  
- Update Student Information  
- Delete Student Records  
- REST API Integration  
- Responsive UI  
- React Hooks usage  

---

## Tech Stack

- React.js  
- tailwindcss  
- Node.js  
- Express.js  
- MySQL  

---

## How to run frontend & backend

npm install  
npm start
